## Twitter Scrapr

 - **Mocha and Chai** for testing.
 - **Express**

*install*
------------
<code> npm install </code>


*development*
------------

<code>npm run dev</code>

In Development mode the express app is starter with nodemon for automatic refresh when do changes.

*production*
------------

<code>npm start</code>

*testing*
-------
<code>npm start<code>
then ->
<code>npm test</code>

For Testing:

 - Mocha
 - Chai
 - Chai-http
